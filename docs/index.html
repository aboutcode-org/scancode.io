Apache licensed

ScanCode.io is Apache-licensed and written in Python with Django and DRF. It has an extensive documentation to help get you started and is supported by an active community of users and adopters.

Made for customization

Scanning pipelines are designed to be easily extended and remixed; they can integrate all and any tools in an elegant unified framework.

Persistent scan data

Backed by a database, ScanCode.io will persist and organize all your scanning projects and make them accessible through a web UI, a JSON REST API or a command line interface.

Because not one scan fits all the codebases, you can select a specific scanning pipeline for each project.

For instance, ScanCode.io provides a best-in-class pipeline to analyse in details a Docker container in a way that is deeply aware of the specific structure of a container image with its layered file systems and combination of system and application packages.

And this will handle equally well Debian, RPM and Alpine Linux distributions, each with their own unique characteristics.

Or you can analyze a complete virtual machine image, or specifically a single application package from Pypi, each with specialized pipelines.
